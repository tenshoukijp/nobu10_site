ＢＧＭ音楽のOVERRIDE

配置場所

OVERRIDE/BGM/*.NBX

ファイル形式は WMA(Windows Media Audio)

蒼天録のBGMフォルダにある BGM/BGM**.NBX は全てフォーマットはただの WMA(Windows Media Audio) です。
ffmpeg などで WMV もしくは WMA として変換して、拡張子だけ .NBX とすれば再生できます。
.WMA への変換は各種一般的な音楽ファイル系の変換アプリケーションや ffmpeg で行える。

ファイル形式変換例

ffmpegでのWMAへの変換例 
ffmpeg -i input.m4a -acodec wmav2 -b:a 320k output.wma
